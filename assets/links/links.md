Projetos Open-Source FPS Recomendados

Repositórios no GitHub permitem estudar o código completo, reproduzir bugs conhecidos (como rocket jumps ou clipping) e modificar para testes locais.

Link do repositório:  https://github.com/id-Software/Quake-III-Arena/tree/master   (Networking multiplayer, physics engine, bunny hopping e wall clipping originais)

Link do repositório:  https://github.com/id-Software/Quake      (Rocket jump como bug inicial; análise de movimento e colisões em C)

Link do repositório: https://github.com/id-Software/DOOM ( Spectre bugs e line-of-sight exploits; base para FPS engines. )

Link do repositório: https://github.com/IMYdev/OpenFPS (FPS experimental em Godot: controller básico, sistema de armas, hitscan e inimigos; ideal para networking e colisões. )

Link do repositório: https://github.com/Richi2293/fpsgame (Feito em Unreal Engine 4 (blueprints/C++): multiplayer dedicado, armas e mapas; estude server-client exploits locais. )

Link do repositório: https://github.com/etlegacy/etlegacy (Baseado em Wolfenstein: Enemy Territory; glitches de movimento acelerado (bunny hop), truques de pron (dive) e lag compensation exploits no netcode.)

Link do repositório: https://github.com/xonotic/xonotic (Sucessor de Nexuiz/Quake-like; rocket jumps intencionais (ex-glitch), wallstrafe e item prediction bugs; estude CPM (Consistent Player Movement). )

Link do repositório: https://github.com/tx00100xt/SeriousSamClassic (Engine Croteam open-source; glitches de escala inimiga, noclipping em mapas e physics desync em multiplayer. )

Link do repositório: (FPS multiplayer em Godot; bugs reportados em hit registration e spawn exploits; ótimo para Godot-specific issues.)

Link do repositório: https://github.com/outplay-net/Fortnite-External (Implementa Aimbot, ESP, Wallhack, Skin Changer e bypass de anticheat. 
Escrito em C++, usa injeção de DLL e técnicas de manipulação de memória.
Bom para estudar DLL Injection, Hooking e interfaces com ImGui)

Link do repositório: https://github.com/belazr/EasyWinHax (Biblioteca C++ para game hacking no Windows.
Oferece funções para Memory Hacking, Pattern Scanning, Hooking, Shellcode Injection. 
Excelente para aprender fundamentos de exploração em processos.)

Link do repositório: https://github.com/std-microblock/blook (Biblioteca moderna em C++ para Inline Hooking e AOB Scanning.
Leve e modular, usada em projetos de segurança e anticheat.
Ótima para entender como interceptar chamadas de função.)

Link do repositório: https://github.com/woldann/NThread (Biblioteca para manipulação de threads sem CreateRemoteThread.
Usada em técnicas avançadas de injeção silenciosa.
Relevante para estudo de exploits de Buffer Overflow e bypass de EDR.)
